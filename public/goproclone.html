GoPro Clone

visit clone at: fauxpro.site

My team cloned GoPro.com, focusing on their e-commerce user flow. My role was to construct the database, which I queried for pertinent data to successfully replicate the cameras and product detail views. I also pair-programed to better understand how to implement the shopping cart which uses Stripe to accept payments. Below are the fully functional views I cloned.
Note: I've revisited this project in May 2018 to implement responsive and mobile friendly design, as well as update our clone to reflect GoPro's current site.
